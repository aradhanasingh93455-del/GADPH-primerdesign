# GAPDH-primerdesign
# GAPDH Primer Design Project

This repository contains the primer design results for the human GAPDH gene (*Homo sapiens* glyceraldehyde-3-phosphate dehydrogenase), generated using NCBI Primer-BLAST.

## Project Overview
The goal was to design highly specific primers for the GAPDH gene, suitable for qPCR applications. 

### Target Details
- **Gene:** GAPDH
- **Organism:** *Homo sapiens*
- **Reference Sequence:** [NM_002046.7](https://www.ncbi.nlm.nih.gov/nucleotide/NM_002046.7)

## Primer Pair 1 Specifications

| Feature | Value |
| :--- | :--- |
| **Forward Primer (5'->3')** | `GTCGCCAGCCGAGCCA` |
| **Reverse Primer (5'->3')** | `CCCAATACGACCAAATCCGTTG` |
| **Product Length** | 76 bp |
| **Forward Tm** | 62.30°C |
| **Reverse Tm** | 59.90°C |
| **GC Content** | 75.00% (Fwd) / 50.00% (Rev) |
